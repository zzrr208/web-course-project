# 使用文档.md
项目简介
本系统为 Python 程序设计课程设计项目，实现了从公开网页爬取电影数据并存储、查询和导出的功能。
系统包括爬虫、数据库存储、数据查询与导出功能，使用 SQLite 作为数据库，数据通过 Wikipedia 获取。

环境要求
操作系统：Windows / Linux / macOS
Python：3.8 及以上版本
依赖库：requests, beautifulsoup4, sqlite3
安装依赖
pip install -r requirements.txt
功能说明
1. 数据爬取
爬虫模块从 Wikipedia 按年份爬取电影信息（如电影名称、URL 和年份），并保存为 JSON 文件。

2. 数据存储
爬取到的电影数据存储到 SQLite 数据库（films.db）。

3. 数据查询
提供交互式查询，用户可以查询指定年份的电影数据及数量。

4. 数据导出
支持将电影数据导出为 CSV 文件，可在 Excel 中打开查看。

系统运行
1. 生成电影数据（爬虫）
运行以下命令启动爬虫，抓取 2000–2024 年的电影数据：
python spider.py
爬取的数据会保存在 films_2000_2024.json 文件中。

2. 导入数据库
运行以下命令将数据导入到 SQLite 数据库：
python db.py
导入完成后，数据会保存在 films.db 文件中。

3. 电影数据查询
查询某一年份的电影数据：
python query.py
根据提示输入年份，如 2024，系统将返回该年份的电影数量及前 20 条电影信息。

4. 导出 CSV 文件
将数据库中的电影数据导出为 CSV 文件：
python export_csv.py
导出的 films.csv 文件包含电影名称、年份、来源和链接。

注意事项
确保运行爬虫后生成了 films_2000_2024.json 文件，然后再运行数据库导入。
查询功能需要手动输入年份。
CSV 导出功能将整个数据库中的数据导出，生成 films.csv。

小结
本系统实现了一个简易的电影数据爬取、存储、查询和导出功能。系统使用 Python 编写，支持命令行操作，能方便地抓取公开数据并进行分析。

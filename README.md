# Python 爬虫与数据库课程设计

## 项目简介
本项目为 Python 程序设计课程设计，主要功能包括：
- 爬取公开网站数据
- 数据清洗与存储（数据库）
- 数据查询与简单统计

## 技术栈
- Python 3
- requests / BeautifulSoup
- SQLite（或 MySQL）

## 运行环境
- Windows / Linux
- Python 3.8+

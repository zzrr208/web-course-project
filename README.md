# Python 爬虫与电影数据分析系统

## 一、项目简介
本项目为 Python 程序设计课程设计项目，主要实现对互联网上**公开电影信息数据**的自动采集、存储、查询与导出。  
系统基于 Python 编写，采用爬虫技术获取数据，并使用 SQLite 数据库存储，支持命令行交互式查询和数据导出。

本项目的数据仅用于课程设计与教学演示，不涉及任何商业用途。

---

## 二、功能概述
系统目前已实现以下功能：

1. **电影数据爬取**
   - 从 Wikipedia 等公开网页按年份爬取电影信息
   - 支持多年份（2000–2024）数据采集
   - 自动去重，累计获取 4800+ 条电影数据

2. **数据存储（SQLite）**
   - 使用 SQLite 作为后端数据库
   - 设计规范的数据表结构
   - 支持唯一约束，避免重复数据

3. **数据查询功能**
   - 支持用户在命令行中输入年份进行查询
   - 显示指定年份的电影数量及部分电影列表
   - 提供交互式查询体验

4. **数据导出功能**
   - 支持将数据库中的电影数据导出为 CSV 文件
   - 导出的 CSV 文件可直接使用 Excel 打开

---

## 三、技术方案与工具
- **编程语言**：Python 3
- **爬虫技术**：requests + BeautifulSoup
- **数据库**：SQLite
- **数据格式**：JSON / CSV
- **运行方式**：命令行程序
- **开发环境**：Windows / Linux / macOS
